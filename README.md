# custom-miniapp

## Project setup

```javascript
// development
npm run serve // 小程序本地开发构建


// production
npm run build // 小程序生产环境构建

```
## 发体验版【默认0.0.0】
```javascript
1. npm run build

2. npm run upload
```

## 发release版【打tag】
```javascript
npm run release
```

## 发release版【复用旧tag，考虑到打了tag时临时又有代码修改】
```javascript
release:override
```
